# Prompt Quizzer – JavaScript Console Quiz

This project is a quiz game created using core JavaScript.  
The quiz runs completely in the browser console using prompt() and alert().

## Features
- Uses array of objects to store questions and answers
- Case-insensitive and whitespace-insensitive input checking
- Score tracking and feedback after every question
- Final score shown at the end

## How to Run
1. Open a browser (Chrome recommended)
2. Open Developer Tools → Console (F12)
3. Copy and paste the contents of quiz.js into the console
4. Press enter to start the quiz
